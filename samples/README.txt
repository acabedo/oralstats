MUESTRAS PARA LA CHARLA DE TUBINGA — Oralstats
================================================

Tres cortes (<= 1 min) de tres entrevistas distintas de PRESEEA-Valencia,
preparados para ejemplificar el proceso de análisis con Oralstats.

Cada muestra es un par WAV + TextGrid con el MISMO nombre base. El TextGrid
conserva la estructura MFA original (compatible con Oralstats y el Etiquetador):
  - "i - words" / "i - phones" / "i - utterances"  (entrevistador)
  - "e - words" / "e - phones" / "e - utterances"  (entrevistado)
Los tiempos están desplazados para empezar en 0 s.

Criterios aplicados:
  - Duración <= 60 s.
  - Participan AMBOS hablantes (entrevistador "i" y entrevistado "e").
  - SIN nombres propios ni datos sensibles (verificado en la transcripción;
    en muestra_1, además, con el tier de anonimización del corpus).
  - Se EXCLUYERON los archivos con intensidad anómala (VAL_H2A ~77 dB,
    VAL_H3A ~48 dB, VAL_H3M ~38 dB).

----------------------------------------------------------------------
muestra_1   (hombre · nivel alto · 20-35 años)
  Duración : 56,2 s   |   Audio: 16 kHz, mono, PCM 16 bit
  Tema     : atracciones de feria, adrenalina, una anécdota
  Hablantes: E = 71 palabras, I = 73 palabras (diálogo muy equilibrado)

muestra_2   (mujer · nivel bajo · >55 años)
  Duración : 56,6 s   |   Audio: 44,1 kHz, mono, PCM 16 bit
  Tema     : el calor, el clima y el cambio climático
  Hablantes: E = 100 palabras, I = 44 palabras

muestra_3   (mujer · nivel medio · 35-55 años)
  Duración : 57,1 s   |   Audio: 44,1 kHz, mono, PCM 16 bit
  Tema     : política y democracia (la antigua Grecia)
  Hablantes: E = 87 palabras, I = 85 palabras (diálogo muy equilibrado)

Las tres celdas cubren los tres tramos de edad y los tres niveles de estudios.
